# Introducción

## Objetivo del proyecto
El objetivo de este proyecto es documentar de forma clara y estructurada una aplicación software,
utilizando Markdown y publicando el resultado como una web mediante GitHub Pages.

## Problema que resuelve
En muchos proyectos el código existe pero no hay documentación accesible ni mantenible.
Este proyecto soluciona ese problema proporcionando documentación técnica versionada,
pública y fácil de consultar.
